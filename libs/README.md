# libs/ — версионируемые библиотеки

Сюда кладём **только перераспространяемые** библиотеки (open-source / собственные),
чтобы они приезжали с репозиторием и их не пришлось искать руками на новой машине.

⚠️ Системные и вендорские библиотеки (`SM3_*`, `System_VisuElems*`, `IoStandard`,
`XJ_*`, EtherCAT и т.п.) сюда класть **нельзя** — лицензии запрещают
перераспространение, и они привязаны к версии CODESYS. Их ставят через
установку CODESYS / Package Manager (см. `../SETUP.md`).

## Что кладём
Файлы библиотек в форматах CODESYS:
- `*.library` — исходная (открывается/правится);
- `*.compiled-library` — скомпилированная (типовая форма распространения).

Текущий кандидат:
- `OSCAT BASIC 3.3.4.0` — выгрузить из репозитория (Tools → Library Repository →
  выбрать OSCAT BASIC → Export) или взять с oscat.de, положить сюда.

## Как установить на новой машине
**Способ 1 — скриптом (рекомендуется):**
```bat
..\scripts\run_install_libs.bat
```
Установит в Library Repository все `*.library` / `*.compiled-library` из этой папки.

**Способ 2 — вручную в CODESYS:**
Tools → Library Repository → Install… → выбрать файл из `libs/`.

После установки `run_build.bat` соберёт проект без ошибок «плейсхолдер не разрешён»
по этим библиотекам.
